This project focuses on analysing heterogeneous datasets (mixed numerical and categorical data) and applying feature transformation techniques to improve machine learning model performance.

The study explores data preprocessing, encoding methods (One-Hot Encoding, Label Encoding), and evaluates multiple ML models to determine the most effective approach for handling categorical variables.

## PROBLEM STATEMENT

Real-world datasets often contain mixed data types (numerical + categorical), which pose challenges for machine learning models.

This project addresses:
- Handling categorical variables effectively
- Preventing information loss during transformation
- Improving model accuracy using encoding techniques

## SOLUTION APROACH

The project follows a structured data analysis pipeline:

1. Data Collection (Kaggle datasets + extracted PDF data)
2. Data Preprocessing (handling missing values, duplicates, outliers)
3. Data Analysis (statistical analysis + visualisation)
4. Feature Transformation (encoding categorical variables)
5. Machine Learning Modelling
6. Model Evaluation (accuracy, confusion matrix, classification report)

## TECH STACK

- Python
- Pandas, NumPy
- Scikit-learn
- Power BI (for visualisation concepts)
- Google Colab
- Tabula (PDF data extraction)

  ## RESULTS
  
- Achieved ~99% accuracy across all models
- Gradient Boosting showed the highest performance (~99.4%)
- Encoding techniques significantly improved model performance
- Identified subject-level performance trends using correlation analysis and visualisation
